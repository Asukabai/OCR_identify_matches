## 项目简介

基于的OCR文本录入识别信息匹配系统

使用了Java的springboot框架，调用了离线版和在线版两种开源OCR接口，实现了前后端分离版本的OCR文本识别与信息内容匹配的功能，并附有前端页面的搜索查询和统计数据的功能
（并提供了一个工具类，安装对应环境之后，可以直接使用本项目下的Python工具类进行运行即可识别并将结果导出到Excel）
